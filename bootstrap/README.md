Experiment 1 — Bootstrap (bootstrap)

Purpose
- Minimal Bootstrap + Vite React experiment demonstrating layout and basic components.

Quick Start
- Install dependencies: `npm install`
- Run development server: `npm run dev`
- Build for production: `npm run build`

Structure
- `index.html` — entry page
- `src/main.jsx` — app bootstrapper
- `src/App.jsx` — main component

Notes
- Use this folder to prototype Bootstrap layouts, grid usage, and component styling.
